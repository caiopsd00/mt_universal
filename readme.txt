Para executar basta executar a instalação do node que pode ser facilitada pelo comando:
$ make

Mas dado que o computador pode estar com alguma configuração diferente ou especifica, recomendamos que tente manualmente instalar o node v18.17.0

Sendo assim, será capaz de executar a maquina padrão desenvolvida com o comando:
$ make run

E caso queira rodar um teste especifico:
$ make run-{numero de teste}

Exemplo: make run-1
